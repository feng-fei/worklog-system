### ISSUE-005: Dashboard加载时控制台出现401/404错误

**严重程度**: Medium  
**类别**: 错误处理  
**控制台错误**:
- `Failed to load resource: the server responded with a status of 401 (UNAUTHORIZED)`
- `Failed to load resource: the server responded with a status of 404 (NOT FOUND)`
- `Error: 登录已过期`

**描述**: 登录后Dashboard加载过程中，部分API请求返回401未授权和404未找到错误。这导致部分Dashboard数据无法加载（如最近收支、活跃客户）。登录流程可能存在token过期或刷新机制问题。

**复现步骤**:
1. 打开系统
2. 输入admin/admin123登录
3. 打开浏览器控制台
4. 观察到401和404错误
